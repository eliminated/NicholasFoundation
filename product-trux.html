Core Features

Everything You Need
In a Social Platform

🔍

Discovery Feed

An intelligent "For You" feed powered by Discovery 1.0 — blending freshness, engagement, and social-graph ranking. Complemented by trending hashtags and suggested users.

✉️

Direct Messaging

Full 1-to-1 conversations with live polling, read receipts, file attachments, edit/unsend windows, and inline moderation reporting via a polished two-panel interface.

🛡️

Moderation Suite

Staff-only moderation with role-based access, report queues, suspicious activity detection, audit logs, archived reports, and admin-only staff access management.

#️⃣

Hashtags & Mentions

Clickable hashtag links with dedicated search filtering, and inline @mention autocomplete. Every mention links directly to the user's profile.

🔔

Notifications

Comprehensive feed for likes, mentions, follows, comments, and replies. Full preferences in Settings, plus user muting to suppress unwanted alerts.

👤

Rich Profiles

Display name, bio, location, website, avatar, and banner. Side-by-side comment docks, bookmarks management, and premium placeholder support.

Security

Built Secure
From Day One

🔒

Prepared Statements

All database queries use PDO prepared statements with emulation disabled — SQL injection is structurally impossible.

🔑

Session Hardening

password_hash / password_verify, session ID regeneration on login, and CSRF protection on every POST form.

📎

Upload Hardening

MIME validation, dimension/size checks, random filenames, image re-decode + re-encode, and .htaccess PHP execution blocking.

✅

Email Verification

Email verification with recognized-domain trust hints, plus linked-account scaffolding for Discord, Google, Facebook, X, and Nicholic Account.
