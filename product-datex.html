Key Features

Spreadsheets
Without Limits

🎨

Canvas-Based Grid

Virtualized rendering with zero DOM-per-cell overhead. Buttery smooth scrolling at 60fps sustained, even with 100,000+ filled cells.

⚙️

WASM-Powered Engine

Formula parsing, dependency tracking, and recalculation run at near-native speed via a Rust engine compiled to WebAssembly. Sub-millisecond recalc on edit.

📐

Formula System

SUM, AVERAGE, MIN, MAX, COUNT, IF, CONCATENATE, and more. Recursive-descent parser with directed acyclic graph dependency tracking and Tarjan's circular reference detection.

📑

Multi-Sheet Workbooks

Create, rename, reorder, duplicate, and delete sheets within a workbook. Full cross-sheet formula references with automatic dependency resolution.

💾

Local Persistence

IndexedDB (browser) and SQLite (desktop) with 3-second autosave under 50ms latency. Operation-log-based undo/redo with unlimited depth and grouped edits.

🖥️

Cross-Platform

Runs everywhere — web browser via Vite + React 19, and native desktop via Tauri 2 (Windows, macOS, Linux), all sharing the same Rust backend.
